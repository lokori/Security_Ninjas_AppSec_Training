This is a web application hacking workshop.

Intended for beginners.




  • The intention is to explore and compromise the target application using OWASP Top 10 vectors. Bruteforcing SSH is not the idea.
  • Targeting someone's laptop or other targets without permission is illegal.
  • You can cheat (after a fashion), but that spoils the fun. On the other hand, being creative is encouraged.
  • Burp, ZAP or some other HTTP proxy is useful, but it's possible to succeed with a normal web browser.
